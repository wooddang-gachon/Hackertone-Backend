# GDGoc Gachon University 해커톤 데이터베이스 명세서

## 테이블 목록
1. [user](#1-user) - 사용자 정보 및 능력치/포인트 관리
2. [temp](#2-temp) - 이메일 OTP 인증 임시 데이터
3. [donation](#3-donation) - 재능 기부 게시글 정보
4. [accept](#4-accept) - 매칭 및 신청 수락 정보
5. [comment](#5-comment) - 게시글 댓글 정보

---

## 1. user
사용자의 계정 정보, 포인트 및 성향/능력치 항목을 저장하는 테이블입니다.

| 컬럼명 | 데이터 타입 | 제약 조건 | 설명 |
| :--- | :--- | :--- | :--- |
| `uidx` | INT | PK (AI) | 사용자 고유 식별자 |
| `gachon_id` | VARCHAR(100) | PK | 가천대학교 아이디 |
| `name` | VARCHAR(100) | - | 사용자 이름 |
| `password` | VARCHAR(100) | - | 비밀번호 (암호화) |
| `point` | INT | - | 보유 포인트 |
| `exercise` | INT | - | 운동 능력치 / 관심도 |
| `study` | INT | - | 공부 능력치 / 관심도 |
| `music` | INT | - | 음악 능력치 / 관심도 |
| `game` | INT | - | 게임 능력치 / 관심도 |
| `clean` | INT | - | 청소 능력치 / 관심도 |

---

## 2. temp
회원가입 시 이메일 OTP 인증을 위한 임시 데이터를 저장하는 테이블입니다.

| 컬럼명 | 데이터 타입 | 제약 조건 | 설명 |
| :--- | :--- | :--- | :--- |
| `tidx` | INT | PK (AI) | 임시 데이터 고유 식별자 |
| `gachon_id` | VARCHAR(100) | PK | 가천대학교 아이디 |
| `otp` | INT | - | 발급된 OTP 인증번호 |

---

## 3. donation
재능 기부 게시글 정보를 저장하는 테이블입니다. 기존의 단일 `tag` 컬럼 대신 개별 카테고리 컬럼을 사용합니다.

| 컬럼명 | 데이터 타입 | 제약 조건 | 설명 |
| :--- | :--- | :--- | :--- |
| `didx` | INT | PK (AI) | 게시글 고유 식별자 |
| `status` | INT | - | 게시글 상태 코드 |
| `writerIdx` | INT | - | 작성자 고유 식별자 |
| `title` | VARCHAR(100) | - | 게시글 제목 |
| `text` | VARCHAR(1000) | - | 게시글 본문 내용 |
| `duedate` | DATE | - | 마감일 |
| `exercise` | INT | - | 운동 카테고리 여부/수치 |
| `study` | INT | - | 공부 카테고리 여부/수치 |
| `music` | INT | - | 음악 카테고리 여부/수치 |
| `game` | INT | - | 게임 카테고리 여부/수치 |
| `clean` | INT | - | 청소 카테고리 여부/수치 |

---

## 4. accept
게시글에 대한 신청 및 매칭 수락 상태를 관리하는 테이블입니다.

| 컬럼명 | 데이터 타입 | 제약 조건 | 설명 |
| :--- | :--- | :--- | :--- |
| `aidx` | INT | PK (AI) | 수락/매칭 고유 식별자 |
| `status` | INT | - | 매칭 상태 코드 |
| `didx` | INT | - | 관련 게시글 고유 식별자 |
| `role` | VARCHAR(100) | - | 역할 구분 |
| `point` | INT | - | 관련 지급/차감 포인트 |

---

## 5. comment
게시글의 댓글 및 대댓글 정보를 저장하는 테이블입니다.

| 컬럼명 | 데이터 타입 | 제약 조건 | 설명 |
| :--- | :--- | :--- | :--- |
| `cidx` | INT | PK (AI) | 댓글 고유 식별자 |
| `parent_cidx` | INT | Null 허용 | 부모 댓글 고유 식별자 (대댓글 구현용) |
| `didx` | INT | - | 관련 게시글 고유 식별자 |
| `uidx` | INT | - | 댓글 작성자 고유 식별자 (API 상에서는 `writeidx`로 참조됨) |
| `text` | VARCHAR(1000) | - | 댓글 본문 내용 |
